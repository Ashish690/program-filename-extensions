# Program Filename Extensions

> Filename extensions of programming languages

| Language      | Source File Extension |
| ------------- |:---------------------:|
| Bash          | .sh                   |
| C             | .c                    |
| C#            | .cs                   |
| C++           | .cpp, .c++, .cc, .c, .cxx, .cpp, .cp |
| Java          | .java                 |
| JavaScript    | .js                   |
| Perl          | .pl                   |
| PHP           | .php                  |
| Python        | .py                   |
| Ruby    			| .rb 					        |
| Simula        | .sim                  |
